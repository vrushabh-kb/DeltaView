# DeltaView

DeltaView is an AI-powered stock analysis platform built using Python, FastAPI, JavaScript, HTML, and CSS.

## Features

- Investor Personality Test
- AI Stock Screener
- Company Financial Analysis
- Revenue & Profit Visualization
- Company Battle Comparison
- Portfolio Dashboard
- DeltaScore Ranking Engine
- AI Investment Insights

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- Chart.js

Backend:
- FastAPI
- Pandas

## Run Project

Backend:

uvicorn backend.main:app --reload

Frontend:

Open index.html using Live Server

Backend URL:
http://127.0.0.1:8000
